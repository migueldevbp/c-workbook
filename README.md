# c-workbook
Material complementario del canal "Miguel Borja Academy". Aquí encontrarás el libro de ejercicios de C++, código fuente, soluciones explicadas y desafíos prácticos diseñados para reforzar el aprendizaje de la programación y desarrollar habilidades de resolución de problemas.

# 📘 Libro de Ejercicios de C++

Bienvenido al repositorio oficial del libro de ejercicios del curso:

🎥 Curso Completo de C++ desde Cero - Miguel Borja Academy

## 🎯 Objetivo

Este repositorio tiene como finalidad complementar el curso mediante ejercicios prácticos, ejemplos resueltos y desafíos de programación que permitan fortalecer el pensamiento lógico y las habilidades de desarrollo en C++.

## 📚 Temas incluidos

- Introducción a C++
- Variables y tipos de datos
- Operadores
- Entrada y salida de datos
- Estructuras condicionales
- Bucles
- Funciones
- Arreglos y matrices
- Strings
- Programación orientada a objetos
- Retos de programación

## 🚀 Metodología

Cada capítulo contiene:

✅ Explicación breve  
✅ Ejercicios propuestos  
✅ Retos adicionales  
✅ Soluciones sugeridas  
✅ Nivel de dificultad
